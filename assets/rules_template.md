## 2. 提取课件（两条腿：文字 + 看图）
```bash
cd "<课程目录>"
PYTHONIOENCODING=utf-8 python <skill>/scripts/extract.py "课件/第N讲.pdf"
```
- 看输出末尾「文字量统计」，对标⚠️的低文字页渲染图片再"看图"：
```bash
... extract.py "课件/第N讲.pdf" --images "img/第N讲" --lowtext-only
```
  然后用 Read 工具逐张看 PNG，把图里信息补进笔记。详见 references/extraction.md。

## 3. 每讲笔记模板
按 `references/open-book.md` 的十节骨架写，文件名 `第NN讲_标题.md`。
带★的速查表（第四~六节）按上面"实体类别"命名；务必写"易混淆/不包含"列。
Word 兼容写法：只用 #/##/###、- 列表、表格、**加粗**。

## 4. 整理完一讲后，更新汇总文件（很重要，别漏）
- `00_总索引.md`：把本讲名词并进对应类别表，同名词只留一行（多讲写多个讲次）。
- `00_思考题汇总.md`：把本讲思考题 + 完整参考答案原样并入。
- 题型专项总表（如 `00_必听歌单.md`）：把本讲专项素材并入。
