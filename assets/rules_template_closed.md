## 2. 提取课件（两条腿：文字 + 看图）
```bash
cd "<课程目录>"
PYTHONIOENCODING=utf-8 python <skill>/scripts/extract.py "课件/第N讲.pptx"
```
- 看输出末尾「文字量统计」，对标⚠️的低文字页渲染图片再"看图"：
```bash
... extract.py "课件/第N讲.pptx" --images "img/第N讲" --lowtext-only
```
  然后用 Read 工具逐张看 PNG，把图里信息（公式、推导图、示意图）补进笔记。
  详见 references/extraction.md。

## 3. 每讲笔记模板（闭卷，覆盖记忆/理解/应用三层）
按 `references/closed-book.md` 的九节骨架写，文件名 `第NN讲_标题.md`：
- 🎯 掌握分级表（★优先级 + 掌握程度"会背/要懂/能用"，依据小测）
- 核心概念卡（是什么 / **为什么·直觉** / 怎么用——重点在"为什么"）
- 公式 & 推导（写全符号含义、**推导主线（能重推）**、易错点，不是只抄结论）
- 方法/算法（步骤 + 每步为什么 + 何时用）、对比辨析（含"何时选"）
- **应用/迁移（场景→选方法+理由+计算，闭卷大题命门，别省）**
- 记忆口诀（只给确实得硬记的项）、自测题（回忆/理解/应用三类）、一句话总结
每讲只产 `第NN讲_标题.md` 一个文件（**本模式不做闪卡、不产 `第NN讲_闪卡.md`**）。
Word 兼容写法：只用 #/##/###、- 列表、表格、**加粗**。
⚠️ 别把闭卷做成纯背诵卡——理解题和应用题才是大题失分重灾区。

## 4. 整理完一讲后，更新汇总文件（很重要，别漏）
> 本模式**不做总表、不做闪卡**：掌握分级靠每讲 ★掌握清单、公式靠每讲"公式 & 推导"节。
> 只维护一个汇总 `00_自测题库.md`。
- `00_自测题库.md`：把本讲自测题（回忆/理解/应用）并进来，答案集中放文件末尾。
